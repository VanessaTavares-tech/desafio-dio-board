# Desafio DIO Board

Este projeto é um **quadro de tarefas simples** desenvolvido em **Java** como parte de um desafio da Digital Innovation One (DIO).

## Funcionalidades

- Adicionar tarefas
- Mover tarefas entre colunas (A Fazer, Em Andamento, Concluído)
- Remover tarefas

## Tecnologias

- Java
- Visual Studio Code
